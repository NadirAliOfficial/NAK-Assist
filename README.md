# NAK Assist

> AI-powered smart assistant for Android — by **Nadir Ali Khan** | [Team NAK](https://teamnak.com)

## What it does

NAK Assist runs as a floating button on any Android app. Tap it to instantly get AI-powered help — write replies, improve text, summarize, translate and more.

## Modes

| Mode | Trigger | Description |
|------|---------|-------------|
| 💬 Smart Reply | Single tap | Reads screen and writes a professional reply |
| 📋 Summarize | Double tap | Summarizes screen content in bullet points |
| ✨ Improve | Long press → menu | Makes text clearer and more professional |
| 🔄 Rewrite | Long press → menu | Rewrites text in a polished way |
| ✂️ Shorten | Long press → menu | Shortens text keeping key meaning |
| ✅ Proofread | Long press → menu | Fixes grammar and spelling errors |
| 💼 Professional | Long press → menu | Rewrites in formal professional tone |
| 😊 Friendly | Long press → menu | Rewrites in warm conversational tone |
| 🌐 Translate | Long press → menu | Translates text to English |
| 📁 Templates | Long press → menu | Pre-built Fiverr response templates |
| ⚙️ Custom | Long press → menu | Type any custom instruction |

## Tech Stack

- **Language:** Kotlin (Native Android)
- **AI Model:** Llama 3.3 70B via Groq API (free)
- **Min SDK:** Android 8.0 (API 26)
- **Permissions:** Accessibility Service, Overlay

## Setup

1. Open in Android Studio
2. Build APK: `Build → Build APK(s)`
3. Install on Android
4. Grant overlay permission + enable accessibility service
5. Tap ⚡ on any screen
