# NAK Assist

AI-powered smart assistant for Android — built for Fiverr reply automation, message improvement, and workflow productivity.

## Features
- AI-powered message drafting and improvement
- Quick reply templates
- Context-aware tone adjustment (formal/casual)
- Fiverr-optimized response suggestions

## Tech Stack
- Kotlin / Android Jetpack
- Local LLM integration
- Material Design 3 UI

## Build
Open in Android Studio and run on device or emulator (API 26+).
